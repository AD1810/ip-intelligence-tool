# 🔥 IP Intelligence Tool PRO

A Python-based GUI tool for IP investigation and threat analysis used in SOC (Security Operations Center) workflows.

---

## 🚀 Features

- 🔍 IP Investigation (ASN & Organization)
- 🚨 Threat Detection using AbuseIPDB API
- 🌐 Nmap Port Scanning
- ⚠️ Smart Risk Classification (Safe / Medium / High)
- 🔁 Multi-IP Scanning Support
- 📁 Automatic Report Generation

---

## 🛠️ Tech Stack

- Python
- Tkinter (GUI)
- Nmap
- AbuseIPDB API

---

## ▶️ How to Run

1. Install dependencies:
```bash
pip install requests
```

2. Install Nmap (if not installed)

3. Run the tool:
```bash
python gui_ip_tool.py
```

---

## ⚠️ Important Note

Before running the tool, add your own API key:

```python
API_KEY = "YOUR_API_KEY_HERE"
```
